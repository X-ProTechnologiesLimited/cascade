Using this template

Here is a package structure:

  • app
  • dist
  • .bowerrc
  • bower.json
  • gulpfile.js
  • package.json

dist - is a folder with ready-to-edit version of the template. Here you simply can open any html or css file in you favorive ediitor and make any changes you want. For the general purpose it is a correct way.

app - is a folder with developer version of the template. This folder contains all source files which used to generate final version of the template. Such as jade folder with .jade files, less folder with .less files and  bower components.

To use this template in a front-end way you should be familiar with some modern web technologies such as node.js, gulp.js, jade and less langs.

Here are some useful links to help you know more about this things:

  • node.js
  • gulpjs
  • jade
  • less

With this links you can find the way to install all of these things on your computer.

So, here are steps to getting started.

  1. Go into the root folder of the template package where app folder locates.
  2. Run the console in this folder.
  3. Run npm install comand in console.
  4. Run bower install comand in console.

In general you have not to be a pro in node.js and gulp.js. You need just to install it. But you need to know something about jade and less. According this, go to app folder, open jade or less files, make your changes and generate ready-to-publish website package. To generate this package you need to run in console gulp dist comand. After th command finished you will have a new dist folder with your new website ready to upload on your server.

Good luck!

                </div>

                some content goes here...

            </div>
        </div>
    </div>
</div>

This structure is about some separated content block with its own section title and subtitle.

This template is based on Bootstrap Framework so you should visit https://getbootstrap.com/ to know more about how to use Bootstrap features.

Background colors

You can change section background color by adding some css classes to it.

<div class="content-section palette-1-bg">

or

<div class="content-section palette-2-bg">

or

<div class="content-section palette-3-bg">

or

<div class="content-section palette-4-bg">

Palette colors you can find in less/vars.less file.

Background images

You can add background image fo the content section. Here is how it works:

<div class="content-section image-bg image-bg-1">

First: add .image-bg class to the section.

Second: add .image-bg-xxx class with css background image.

That is all.
